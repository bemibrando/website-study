## Overview
---
### <b>The Challenge</b>
- Create an API that will be able to add a new band (only validate, this API doesn't record) and return all bands, bands by ID and bands by genre.

<br />

### <b>Project Composition</b>
- Get /bands
- Get /bands/:uuid
- Post /bands/store
- Get /bands/genre/:genre
